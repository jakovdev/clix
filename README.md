## [GitHub](https://www.github.com/jakovdev/clix)

Single-header file CLI libraries for C

## The list:
- [args.h](https://www.github.com/jakovdev/clix/blob/main/args.h) - Decentralized CLI argument framework.
- [print.h](https://www.github.com/jakovdev/clix/blob/main/print.h) - WIP
- [progress.h](https://www.github.com/jakovdev/clix/blob/main/progress.h) - WIP

## How to Use

Simply copy-paste any header you want to your C project.

In your main source file define `{HEADER_NAME}_IMPLEMENTATION` and then include it, for example:

```c
#define ARGS_IMPLEMENTATION
#include "args.h"
```
